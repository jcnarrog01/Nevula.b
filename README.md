# Nevula

Nevula is a travel-inspired apparel website where customers can explore a clothing drop, view featured products, and see a destination countdown connected to the brand's ticket-based travel giveaway experience.

## Technologies Used

- HTML
- CSS
- JavaScript

## Pages Included

- Landing page
- Products page
- Countdown page

## Main Features

- Responsive multi-page website
- Immersive animated landing page
- Travel-inspired brand experience
- Featured product sections
- Product page with size selection
- Simulated add-to-cart interaction
- Ticket calculation based on `$1 spent = 1 ticket`
- New York countdown page
- FAQ accordion
- Mobile navigation
- Scroll reveal animations
- Consistent header and footer across pages

## Project Scope

This project is a front-end brand website focused on UI, visual identity, responsiveness, and interactive user experience. The current version emphasizes strong visual presentation, clear brand storytelling, and polished client-side interactions using only HTML, CSS, and JavaScript.

## What I Learned

- Building a multi-page website with a clear navigation flow
- Creating a consistent design system across several pages
- Using brand colors and visual assets to support a stronger identity
- Implementing JavaScript interactions for navigation, countdowns, accordions, and cart simulation
- Improving responsive layouts for desktop and mobile experiences
- Creating a stronger visual identity for a fashion-focused brand
